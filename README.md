# Калькулятор себестоимости блюда

Простой веб-калькулятор для ресторанного бизнеса.  
Рассчитывает итоговую цену блюда на основе себестоимости продуктов и наценки.

## Технологии
- HTML5
- CSS3
- JavaScript (ES6)

## Как запустить локально
1. Скачайте репозиторий
2. Откройте `index.html` в браузере
